# Trebuchet Simulator

## Add package
```
] add https://github.com/Roboneet/Trebuchet.jl
```
## Running example
```
using Trebuchet
# using Blink # if not on Juno or IJulia
using Pkg
cd(Pkg.dir("Trebuchet"))
include("example.jl")
```
